## 💡 About

Hero Rescuers is an old side project I experimented with. It's an online first responder game that supports up to 50 concurrent users. This was mostly to test the tech listed below.

## 👨‍💻 Tech

- Unity
- Mirror
- LazyECS
- LazyECS.Networking
- HDRP
- RTX
- NodeJS
- Jenkins
- Docker
